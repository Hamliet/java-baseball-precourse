# 숫자 야구 게임
---

### 구현할 기능 목록

1. 컴퓨터가 숫자(정답)을 생성한다.
    - 숫자 범위는 1~9이다.
    - 중복된 숫자가 나오면 안된다.

2. 사용에게 정답을 입력받고 정답을 확인한다.
    - 정답과 사용자가 입력한 값을 비교해 스트라이크, 볼의 갯수를 카운트한다.
    - 스트라이크,볼, 낫싱을 출력해준다.
    - 정답을 맞추지 못한 경우 사용자에게 다시 정답을 입력받는다.
    - 정답을 맞춘 경우 해당 라운드를 종료한다.

3. 사용자가 게임을 다시 시작할지 확인한다.
    - 다시시작(1), 종료(2)가 아닌 다른 값이 입력되면 올바른 입력을 기다린다.
